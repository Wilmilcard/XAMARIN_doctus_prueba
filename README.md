# Doctus App
App Xamarin para prueba de ingreso de Doctus

## Tecnologias usadas
- Xamarin Forms
- Visual Studio 2019
- C#
- SQL Lite

## Ejecucion:
### 1. Clonar el reporsitorio en Visual Studio (preferiblemente 2019)
### 2. Compilar la app desde un dispositivo Android (Version 9 o 10)
